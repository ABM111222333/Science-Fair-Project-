Project Overview

This project demonstrates a fully functional fire detection system built around the Arduino microcontroller platform. The system continuously monitors environmental temperature and triggers immediate multi-sensory alarms when hazardous conditions are detected.

The system operates on a threshold-based detection principle, activating at 50Â°C (122Â°F) â€“ a critical temperature indicating potential fire hazards. It combines hardware integration with efficient software logic to create a reliable safety device.

System Components

The fire alarm system integrates four key hardware components:

Temperature Sensor

Analog heat sensor (A0 pin) that converts thermal energy to electrical signals. Provides real-time temperature data to the microcontroller.

Piezoelectric Buzzer

Audible alarm (Pin 9) emitting 1000Hz warning tone during activation. Provides immediate audible notification of danger.

LED Indicator

Visual alert (Pin 13) - Built-in Arduino LED that illuminates during alarm state for visual confirmation.

Arduino Uno

Microcontroller brain that processes sensor data, makes decisions, and controls all outputs based on programmed logic.

How It Works

The system follows a precise operational cycle that repeats every second:

Temperature Reading

Analog sensor reads voltage (0-5V) corresponding to temperature

Data Conversion

Analog value (0-1023) converted to Celsius using scaling formula

Threshold Comparison

Compares measured temperature against 50Â°C safety threshold

Alarm Activation

If temperature > 50Â°C: Activates LED and 1000Hz buzzer tone

Safety Feature: The system automatically deactivates alarms when temperature falls below the threshold, providing continuous protection without manual reset.

Core Algorithm

The system's intelligence comes from this Arduino code that implements the detection logic:

// ===== PART 1: Pin Definitions & Global Variables =====
const int heatSensorPin = A0; // Pin for heat sensor
const int buzzerPin = 9; // Pin for buzzer (PWM capable)
const int ledPin = 13; // Pin for LED indicator
const float thresholdTemp = 50.0; // Alarm trigger at 50Â°C

// ===== PART 2: setup() Function =====
void setup() {
Serial.begin(9600); // Start serial communication
pinMode(buzzerPin, OUTPUT);
pinMode(ledPin, OUTPUT);
}

// ===== PART 3: loop() Function - Main Program =====
void loop() {
// 1. Read temperature sensor
int sensorValue = analogRead(heatSensorPin);
// 2. Convert to temperature
float temperature = sensorValue * (5.0 / 1023.0) * 100.0;
// 3. Send to Serial Monitor for monitoring
Serial.print("Temperature: ");
Serial.println(temperature);
// 4. Check threshold and trigger alarm if needed
if (temperature > thresholdTemp) {
// TRIGGER ALARM
digitalWrite(ledPin, HIGH);
tone(buzzerPin, 1000); // 1000 Hz tone
} else {
// SYSTEM SAFE
digitalWrite(ledPin, LOW);
noTone(buzzerPin);
}
delay(1000); // Wait 1 second
}
