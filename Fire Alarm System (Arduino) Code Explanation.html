Project Overview

This project implements a simple yet effective fire alarm system using an Arduino. It continuously monitors temperature using a heat sensor and triggers a visual (LED) and audible (buzzer) alarm when a dangerous threshold is exceeded.

Complete Arduino Code

The complete, functional code for the system is provided below.

// Fire Alarm System - Arduino Code
// Team 7-Q

// ===== PART 1: Pin Definitions & Global Variables =====
const int heatSensorPin = A0;  // Pin for the heat sensor (Analog Input)
const int buzzerPin = 9;      // Pin for the buzzer (PWM capable)
const int ledPin = 13;        // Pin for the LED (Built-in LED on most boards)

const float thresholdTemp = 50.0; // Alarm trigger at 50Â°C

// ===== PART 2: setup() Function - One-Time Setup =====
void setup() {
    // Start serial communication for debugging
    Serial.begin(9600);

    // Set buzzer and LED pins as OUTPUT
    pinMode(buzzerPin, OUTPUT);
    pinMode(ledPin, OUTPUT);
}

// ===== PART 3: loop() Function - Main Program (Repeats Forever) =====
void loop() {
    // 1. Read the heat sensor value
    int sensorValue = analogRead(heatSensorPin);

    // 2. Convert sensor reading to temperature (Example conversion)
    float temperature = sensorValue * (5.0 / 1023.0) * 100.0;

    // 3. Print temperature to the Serial Monitor
    Serial.print("Temperature: ");
    Serial.println(temperature);

    // 4. Check if temperature exceeds the threshold
    if (temperature > thresholdTemp) {
        // TRIGGER ALARM: Turn on LED and sound buzzer
        digitalWrite(ledPin, HIGH);
        tone(buzzerPin, 1000); // 1000 Hz tone
    } else {
        // SYSTEM SAFE: Turn off LED and buzzer
        digitalWrite(ledPin, LOW);
        noTone(buzzerPin);
    }
    delay(1000); // Wait for 1 second before next reading
}
Note: The temperature conversion formula sensorValue * (5.0 / 1023.0) * 100.0 is an example. You may need to calibrate it based on the specific heat sensor module you are using.
